# 🚀 アルゴリズム実戦解説ガイド
(Algorithm Deep Dive: Theory to Code)

このリポジトリに実装されている主要なアルゴリズムの仕組みと、コードのどこに注目すべきかを解説します。

---

## 1. 量子センシング (Quantum Sensing)

### **反復位相推定 (IPE / IQPE)**
- **場所**: `src/sensing/core.py` -> `iterative_phase_estimation`
- **何をしている？**: 1つの補助ビットを「使い回し」て、未知の角度を小数点以下まで 1ビットずつ突き止めます。
- **読み解きのヒント**: `reversed(range(num_bits))` のループに注目。下の桁（精度）から順番にパズルを解いていく様子がわかります。

### **フィードバック安定化 (PID制御)**
- **場所**: `src/sensing/control.py` -> `PIDController`
- **何をしている？**: 周囲のノイズでセンサの向きがズレたとき、それを自動で引き戻します。
- **読み解きのヒント**: `Kp`, `Ki`, `Kd` という3つのパラメータが、それぞれ「今のズレ」「溜まったズレ」「ズレる勢い」を抑える役目をしています。

---

## 2. 量子最適化 (Quantum Optimization)

### **量子断熱計算 (AQC)**
- **場所**: `src/optimization/core.py` -> `AdiabaticOptimizer`
- **何をしている？**: 「簡単なパズル」から始めて、ゆっくりと時間をかけて「難しいパズル（問題）」に変形させます。断熱定理により、ゆっくり変えれば答えが維持されます。
- **読み解きのヒント**: `build_circuit` 内の `for step in range(N)` ループが、時間を少しずつ進めている部分です。

### **QAOA (Quantum Approximate Optimization Algorithm)**
- **場所**: `src/optimization/qaoa.py`
- **何をしている？**: AQCのショートカット版です。量子コンピュータで試行錯誤し、古典コンピュータが「もっと良くするには？」と指示を出します。

---

## 3. 量子通信 (Quantum Communication)

### **BB84プロトコル**
- **場所**: `src/communication/core.py`
- **何をしている？**: 盗聴者が光子を覗くと「状態が変わってしまう」という物理法則を使い、安全な鍵を作ります。
- **読み解きのヒント**: `analyze_qber` 関数で「どれだけエラーが出ているか」を計算し、盗聴を検知しています。

---

## 4. 量子機械学習 (Quantum machine Learning)

### **量子カーネル法 (Quantum Kernels)**
- **場所**: `src/ml/kernel.py`
- **何をしている？**: 古典コンピュータでは分類が難しい複雑なデータを、量子ビットが作る広大な「世界（特徴空間）」に飛ばして、スッキリと分類します。
- **読み解きのヒント**: `calculate_kernel_element` で、2つのデータの「量子的な似具合」を計算しています。
