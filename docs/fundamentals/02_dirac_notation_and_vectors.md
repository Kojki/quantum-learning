# Step 02: ブラケット記法とベクトル表現（確率振幅の数学）

Step 01では「30%の0と70%の1が混ざっている状態」と直感的に表現しました。
これを数学的に正確に、かつ計算しやすい形で記述するためのルールが **ブラケット記法 (Dirac Notation)** と **ベクトル** です。

---

## 1. 状態をベクトルで表す
量子ビットの状態は、「縦に並んだ数字の組（列ベクトル）」として表現します。
基本となる `0` と `1` を次のように定義します。

- **$|0\rangle$ (ケト ゼロ)**: 「100%確実に0が出る状態」
  $$ |0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix} $$
- **$|1\rangle$ (ケト イチ)**: 「100%確実に1が出る状態」
  $$ |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix} $$

この `| \rangle` という記号を **ケット (Ket)** と呼びます。中に入っている文字は単なる「名前ラベル」です。

## 2. 重ね合わせを数式にする
量子ビットが「回転しているコイン」のように重なり合っている状態を $|\psi\rangle$ (プサイ) と名付けます。
これは $|0\rangle$ と $|1\rangle$ の「足し算」で書けます。

$$ |\psi\rangle = \alpha |0\rangle + \beta |1\rangle $$

ベクトルで書くとこうなります。
$$ |\psi\rangle = \alpha \begin{pmatrix} 1 \\ 0 \end{pmatrix} + \beta \begin{pmatrix} 0 \\ 1 \end{pmatrix} = \begin{pmatrix} \alpha \\ \beta \end{pmatrix} $$

ここでの $\alpha$ (アルファ) と $\beta$ (ベータ) を **確率振幅 (Probability Amplitude)** と呼びます。

## 3. なぜ「確率」ではなく「確率振幅」なのか？
$\alpha$ や $\beta$ は「単なる確率 (%)」ではありません。**複素数**（例: $0.5 + 0.3i$ や $-0.8$）です。
ここが古典的な確率論と決定的に異なる点です。マイナスの確率なんてあり得ませんが、マイナスの「波の振幅」はあり得ます（波の谷の部分）。

### 測定確率のルール（ボルンの規則）
私たちが測定して得られる「確率」は、この振幅を **「2乗絶対値」** したものになります。

- 測定して `0` が出る確率 = $|\alpha|^2$
- 測定して `1` が出る確率 = $|\beta|^2$

### 規格化条件（確率の合計は必ず1）
コインを投げれば、必ず表か裏のどちらかが出ます。途中でコインが消滅することはありません。
つまり、確率の合計は必ず 1 (100%) にならなければなりません。

$$ |\alpha|^2 + |\beta|^2 = 1 $$

この条件を「規格化条件 (Normalization)」と呼びます。量子計算の旅の間中、ベクトルは常にこのルールに従って長さを「1」に保ち続けます。

---

## 4. 有名な重ね合わせ状態
よく出てくる状態として、**$|+\rangle$ (プラス状態)** があります。これは「0と1が等確率（50%ずつ）で混ざっている状態」です。

$$ |+\rangle = \frac{1}{\sqrt{2}} |0\rangle + \frac{1}{\sqrt{2}} |1\rangle = \begin{pmatrix} \frac{1}{\sqrt{2}} \\ \frac{1}{\sqrt{2}} \end{pmatrix} $$

検証してみましょう。
- 0になる確率 = $|\frac{1}{\sqrt{2}}|^2 = \frac{1}{2} = 50\%$
- 1になる確率 = $|\frac{1}{\sqrt{2}}|^2 = \frac{1}{2} = 50\%$

見事に合計100%になりますね！
