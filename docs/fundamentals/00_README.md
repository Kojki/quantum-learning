# Fundamentals：量子計算の基礎

---

## このセクションについて

このセクションでは、量子コンピュータを理解するために必要な基礎理論を整理していきます。

量子コンピュータは単なる高速計算機ではありません。  
古典計算とは異なる物理法則に基づいた計算モデルです。

そのため重要なのは、

- 状態をベクトルとして捉えること
- 数式による表現に慣れること
- 古典計算との構造的な違いを理解すること

です。

直感的な説明から入り、徐々に数式へと進んでいきます。
