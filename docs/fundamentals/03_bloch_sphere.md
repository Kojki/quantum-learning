# Step 03: ブロッホ球での視覚化（位相の概念）

Step 02で、量子状態 $|\psi\rangle$ は複素数 $\alpha, \beta$ を用いて次のように書けることを学びました。
$|\psi\rangle = \alpha |0\rangle + \beta |1\rangle$ （ただし $|\alpha|^2 + |\beta|^2 = 1$）

確率の合計が「1」になるということは、特定の制約があるということです。これを図形的に捉え直したものが **「ブロッホ球 (Bloch Sphere)」** です。

---

## 1. ブロッホ球とは？
ブロッホ球は、1つの量子ビットが取り得るすべての状態を **「半径1の球の表面」** にマッピングした3Dのマップです。

### 基本的な方角
- **北極 (Z軸のプラス)**: $|0\rangle$ の状態（確実に0が出る）。
- **南極 (Z軸のマイナス)**: $|1\rangle$ の状態（確実に1が出る）。
- **赤道上の点 (XY平面)**: 0と1が 50% vs 50% で重なり合っている状態（$|+\rangle$ など）。

量子ビットの状態 $|\psi\rangle$ は、必ずこの球の表面のどこか1点を指す「矢印（ベクトル）」として表現できます。

## 2. 極座標での新しい書き方
球の表面上の点は、2つの「角度」を使って表すことができます。
1. **$\theta$ (シータ)**: 北極からどれだけ傾いているか。(緯度)
2. **$\phi$ (ファイ)**: 赤道上でどの方向を向いているか。(経度)

これを使うと、状態 $|\psi\rangle$ は次のような美しい式に書き換えられます。

$$ |\psi\rangle = \cos\left(\frac{\theta}{2}\right) |0\rangle + e^{i\phi} \sin\left(\frac{\theta}{2}\right) |1\rangle $$

難しく見えますが、要素を分解すると非常にシンプルです。

### $\theta$ (シータ) の意味：確率のシーソー
$\theta$ は、「0と1がどちらの割合で混ざっているか」を決めます。
- $\theta = 0^\circ$ (北極) のとき: $\cos(0)=1, \sin(0)=0$ $\to$ $|0\rangle$ のみ！
- $\theta = 180^\circ$ (南極) のとき: $\cos(90^\circ)=0, \sin(90^\circ)=1$ $\to$ $|1\rangle$ のみ！
- $\theta = 90^\circ$ (赤道) のとき: $\cos(45^\circ) = \frac{1}{\sqrt{2}}, \sin(45^\circ) = \frac{1}{\sqrt{2}}$ $\to$ まさに $|+\rangle$ の状態（確率50%ずつ）。

### $\phi$ (ファイ) の意味：位相（Phase）
ここが量子力学のキモです。$e^{i\phi}$ はオイラーの公式による「複素数平面での回転」を表します。
つまり、$\phi$ が変わっても $|0\rangle$ や $|1\rangle$ が観測される**確率論的な割合（割合＝振幅の絶対値の2乗）は一切変わりません。**

では $\phi$ は何のためにあるのか？
それは **「量子干渉（波の強め合い・弱め合い）」を起こすためのタイミング（波のズレ）** を保存しているのです。
（センシングアルゴリズムである「ラムゼー干渉計」や「反復位相推定 (IPE)」は、まさにこの赤道上の $\phi$ の角度を精密に測る技術です）

## 3. なぜ視覚化が重要なのか？
数式だけで $\alpha$ と $\beta$ を追跡するのは人間にとって困難です。
しかしブロッホ球を使えば、「量子ゲート（回路の命令）」とは単なる **「ブロッホ球上での矢印の回転」** に過ぎないことが直感的に理解できます。

例えば、「Xゲート」と呼ばれるものは「X軸を串にして、180度回転させる」という操作です。北極にある $|0\rangle$ は、180度回って南極の $|1\rangle$ になります。
