# Step 07: 量子もつれとCNOTゲート（ベル状態）

量子コンピュータの最大の武器であり、アインシュタインが「不気味な遠隔作用」と呼んだのが **「量子もつれ (Entanglement)」** です。
ここでは、「もつれ」を回路でどう作るのか、そして数式でどう表されるのかを学びます。

---

## 1. 2ビットの魔法のゲート：CNOT
量子もつれを生み出すために欠かせないのが、2量子ビットを直接操作する **CXゲート（Controlled-NOT / CNOTゲート）** です。

CNOTには、指示を出す「コントロールビット」と、操作される「ターゲットビット」があります。
- **ルール**: コントロールビットが `1` のときだけ、ターゲットビットに `X` ゲート（反転）をかける。コントロールが `0` のときは何もしない。

| 実行前の状態 | コントロール | ターゲット | CNOT後の状態 |
| :--- | :---: | :---: | :--- |
| $|00\rangle$ | 0 | 0 | $|00\rangle$ (変化なし) |
| $|01\rangle$ | 0 | 1 | $|01\rangle$ (変化なし) |
| $|10\rangle$ | 1 | 0 | **$|11\rangle$ (ターゲット反転)** |
| $|11\rangle$ | 1 | 1 | **$|10\rangle$ (ターゲット反転)** |

古典計算の `XOR`（排他的論理和）と似た働きですが、量子計算ではこれを「重ね合わせ状態」に対しても同時に実行できるのが大きな違いです。

## 2. ベル状態の作成（もつれの発生）
では、コントロールビットを「重ね合わせ状態」にして、CNOTをかけてみましょう。これが魔法の瞬間です。

**[Step 1]** 1つ目のビット（コントロール）にHゲートをかけ、重ね合わせ（$|+\rangle$）にします。
状態は $\frac{1}{\sqrt{2}}|0\rangle + \frac{1}{\sqrt{2}}|1\rangle$ になり、全体としては $\frac{1}{\sqrt{2}}|00\rangle + \frac{1}{\sqrt{2}}|10\rangle$ になります。

**[Step 2]** この状態に対して、ターゲットを2つ目のビットとした CNOT ゲートをかけます。

- 第1項の $|00\rangle$ $\to$ コントロールが `0` なので、そのまま $|00\rangle$。
- 第2項の $|10\rangle$ $\to$ コントロールが `1` なので、ターゲットが反転して $|11\rangle$ になる！

**[結果]**
得られた最終状態は次のようになります。
$$ |\Phi^+\rangle = \frac{1}{\sqrt{2}}|00\rangle + \frac{1}{\sqrt{2}}|11\rangle $$

この状態を **ベル状態 (Bell State)** と呼びます。これが「もつれ状態」のもっとも純粋な形です。

## 3. もつれの何がすごいのか？
ベル状態 $\frac{1}{\sqrt{2}}|00\rangle + \frac{1}{\sqrt{2}}|11\rangle$ の特徴を見てみましょう。
ここには、$|01\rangle$ や $|10\rangle$ といった「バラバラの状態」が存在していません。

もし、この2つの量子ビットを何万キロも引き離し、アリスとボブにそれぞれ1つずつ持たせたとします。
1. アリスが手元のビットを**測定**します。
2. もし結果が `0` ならば、波は $|00\rangle$ に収縮します。すると、何万キロ離れていようとも、ボブのビットは測定せずとも**瞬時に（100%確実に）`0` であることが確定**します。
3. アリスの測定結果が `1` ならば、ボブの手元も瞬時に `1` と確定します。

2つのビットは完全に「運命共同体」になっているのです。（この性質を利用したのが「量子テレポーテーション」です）

### 数学的な「もつれ」の定義
もつれていない状態（単なる独立したビットの集まり）は、必ず「ビット1の状態 $\otimes$ ビット2の状態」という形に因数分解できます。
しかし、ベル状態はどう頑張っても **2つのベクトルのテンソル積形に分解できません**。数学的に因数分解できない状態＝もつれ状態、と定義されます。
